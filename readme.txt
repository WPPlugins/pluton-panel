Add meta boxes for your posts, pages, posts types and Theme Panel to extend the functionality of Pluton.

== Description ==

This plugin requires the [Pluton](https://plutonwp.com/) theme to be installed.

== Installation ==

1. Upload `pluton-panel` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done!

== Frequently Asked Questions ==

= I installed the plugin but the panel do not appear =

This plugin will only work with the [Pluton](https://plutonwp.com/) theme.
